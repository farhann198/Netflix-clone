# 🎬 Netflix Clone (HTML & CSS)

## 📌 Overview

This project is a **Netflix landing page clone** built using only **HTML and CSS**.

It replicates the look and feel of the original Netflix homepage, focusing on layout, styling, and responsiveness.

---

## ✨ Features

* 🎯 Clean and modern UI inspired by Netflix
* 📱 Responsive design (works on mobile, tablet, desktop)
* 🎨 Styled using pure CSS (no frameworks)
* 🖼 Hero section with background image
* 📺 Movie/TV show sections with cards
* 🔻 Footer similar to Netflix

---

## 🧠 What I Learned

* Structuring web pages using semantic HTML
* Styling complex layouts with CSS
* Flexbox and Grid for responsive design
* Positioning, overlays, and gradients
* Creating real-world UI clones
